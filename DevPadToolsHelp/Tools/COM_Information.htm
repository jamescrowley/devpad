COM Information

The COM Information window allows you to browse the available classes and Enums in a ActiveX DLL, OCX or TLB file. The UI is very similar to the Visual Basic Object Browser.

 

To load information from a COM component, click Select... and choose the file. The information will then be loaded. You can then select a class from the left hand pane, and its members will be displayed. If you select an individual member, its parameter information will be displayed.

 

The parameter information is currently only available in VB format. If you are interested in expanding this functionality to C++/Java, please email james@vbweb.co.uk
