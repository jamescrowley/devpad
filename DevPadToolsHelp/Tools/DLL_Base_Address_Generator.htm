DLL Base Address Generator

Every DLL or ActiveX control has a Base Address. This specifies the default address in the memory into which your control is loaded.

 

In Visual Basic, you can set this by going to Project|Properties|Compile. This DLL Base Address Generator allows you to randomly generate a base address for your controls. It also keeps a history list of addresses you have already used to prevent duplication.

 

To generate a DLL Base Address, click the Generate button. You can then press Ctrl+C to copy the address to the clipboard. You will be prompted as to whether you want the address added to the history list.

 

To add an Address to the history list, click Add. The Address in the text box will be added to the list.

 

To remove an Address, select it from the list, and click the Delete button.
