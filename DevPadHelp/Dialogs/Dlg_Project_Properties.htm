Project Properties Dialog

The Project Properties dialog provides information on the project.

 

  • Project Path - The filename of the project

  • Project Root - The root path of the project, where the relative paths will be taken from.

  • Project Name - The name of the project.

  • Author - The author of the project. You can specify a default Project author in the Options dialog.

  • Description - A short description of the project.

 

To display the dialog click Project|Properties
