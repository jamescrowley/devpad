Project Window

The project window allows you to organise your Programming Projects.

 

You can currently add links to

 

  • Files

  • Static Folder (link to Explorer window)

  • Live Folder (explore in Project Window)

  • E-Mail

  • Internet URL

 

as well as creating an unlimited number of folders to organise your project.

 

Note: Developers Pad projects use relative paths rather than absolute paths where possible. The root path from which this is calculated can be changed in the Project Properties dialog.
