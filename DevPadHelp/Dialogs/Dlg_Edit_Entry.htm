Edit Entry Dialog

The Edit entry dialog allows you to edit existing items in your Code list, and create new ones.

 

  • Name - The name of the entry. This will be displayed in the Insert Text window.

  • After Text - Check this box if you want the cursor to be placed after the text when inserting it into the document.

  • Cursor Pos - This allows you to specify the position of the cursor once the text has been inserted into the document. You can enter a number, or use the up/down arrows to change this value.

  • Length - This sets the length of the selection. You can use this feature to make a certain area of text in the entry be automatically selected when inserted. You can enter a number, or use the up/down arrows to change this value.

  • Entry - Enter the text for this entry here. You may also use the constants created in the Options dialog. The pre-set constants are:

    • %date - The current date

    • %time - The current time

    • %ftitle - The filename of the document

    • %path - The path of the document

    The format of these constants can be edited in the Options dialog.
