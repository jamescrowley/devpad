Font Name - The font which is used to display non-RTF documents in Developers Pad

Font Size - The size of the font to use
Text Wrapping - The text wrapping mode:

  • No Wrap - Text is not wrapped

  • WYSIWYG - What you see is what you get. The text will be wrapped according to your printer settings and paper size

  • Word Wrap - The text is wrapped to fit the screen size

Default Project Name - The default name of the project to use

Default Author - The default author name to use in a Developers Pad Project

Reload Last Project - Whether the last loaded project should be re-loaded automatically on startup

Project Window Is Dockable - Whether the project window can be docked

Constants - This allows you to specify constants for use in the Insert Code window.  The pre-set constants are:

  • %date - The current date

  • %time - The current time

  • %ftitle - The filename of the document

  • %path - The path of the document

You can edit the format by selecting the item and adjusting the Value property. The file storing the constants is %DevPadPath%\constants.txt.

This allows you to add, delete and modify the templates available for use in Developers Pad. Each item can have a default language specified. These files are stored in %DevPadPath%\_templates

Show New Tab - Whether the New tab should be displayed in the Open dialog

Remember Last Location - Whether Developers Pad should return to the last folder you opened a file from when displaying the Open dialog

Favourites - This allows you to edit the favourite folders listed in the Open dialog. The file storing this information is in %DevPadPath%\favourites.txt

Use temporary file where possible - Whether the preview dialog should save a temporary file instead of prompting to save. This is not available for server-side scripts.

Prompt to save for server files - Should Developers Pad prompt you to save a modified server-side script?

Server File Extensions - The extensions for files that should be treated as server-side scripts (and therefore need to be viewed from a server rather than the hard disk)

Local Path - The local paths for the server, so Developers Pad knows which server to use. Multiple locations can be listed, seperated by a ;

Server - The servers available for use. Multiple servers can be listed, seperated by a ; (which will be mapped to the corresponding item in Local Path)

Show Tips at Startup - Display the tips dialog at startup?

Allow more than once instance - If this option is checked, a new instance of Developers Pad will be opened each time you run DevPad.exe. If it is not checked, a previous instance will be activated.

Check DevPad is default viewer  - Whether Developers Pad should check if it is default viewer for the extensions listed below.

VB Path - The path of Visual Basic. This is used for the Tools|VB Commands menu item.

Options Dialog

The options dialog allows you to change configuration settings for Developers Pad. You can display the dialog by clicking Tools|Options...
