Features

Developers Pad 1 features...

 

  • Syntax Highlighting

  • Fully customizable language syntax files

  • Unlimited Undo/Redo

  • Auto-Indent code as you type

  • Find & Replace multiple lines of text, match case, search from cursor, find all, and more!

  • Find & Replace Logs

  • Insert code - with full control over cursor position, selection length and variable inserts such as filename.

  • Go forward/back to recent lines you have been editing

  • Full support for Add-Ins, including an Icon Editor, Class Spy, Base Address Generator, and more as standard.

  • Favourite Folders for Open and Save

  • Recent File History

  • Customizable Templates

  • No-save Browser Preview, and support for server-side language previews.

  • Manage your files using Developers Pad Projects, or open your existing VB project!
