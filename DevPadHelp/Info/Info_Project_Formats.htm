Project Formats

Developers Pad Projects are saved in XML format with a .dpp extension.

 

Apart from the Developers Pad Project format, Developers Pad can also open the following project formats:

 

  • Visual Basic Projects & Groups

  • Visual C++ Projects

  • Visual Studio.NET Projects

 

Note: Currently, you can only save a project in the Developers Pad format. If you wish to make changes to the other formats, you must make the changes manually.
